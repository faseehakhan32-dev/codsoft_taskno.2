# 🧮 Two Number Calculator (Web App)

A simple web-based calculator built using **HTML, CSS, and JavaScript** that performs basic arithmetic operations on two numbers.

---

## 🚀 Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Simple and clean user interface
- Instant result display

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

---

## 📂 Project Structure
```
calculator/
│── index.html
│── style.css
│── script.js
│── README.md
```

---

## ▶️ How to Run

1. Clone the repository:
```
git clone https://github.com/your-username/calculator.git
```

2. Open the project folder:
```
cd calculator
```

3. Run the app:
- Open `index.html` in any web browser

---

## 💡 Usage
1. Enter the first number  
2. Enter the second number  
3. Select an operation (+, -, *, /)  
4. View the result  

---

## 📌 Example
```
Input: 10 and 5  
Operation: +  
Output: 15
```

---

## ⚠️ Notes
- Division by zero is not allowed  
- Please enter valid numbers only
